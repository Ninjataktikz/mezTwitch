July 3, 2026

the ledger
updates itself.

Every number on the broadcast now comes from the FACEIT API. Manual tweaks remain as fallbacks only. This page documents what runs itself and the two config recipes that finish the live layer.

01 — what updates automatically (refresh: every 5 min)
elo · k/d · hs%
Scenes 01, 02, 03, 04 + the ledger stat post. Live from FACEIT; sample numbers show only if the API is unreachable.
session delta
"+31" next to elo. Baseline = your elo at the first scene load of each day; resets automatically at the next stream day.
today w—l
Counted from today's FACEIT match history. Shows on the 01/04 baselines and the 02 stat strip.
last map
"last map — ancient 13:7 w" on the starting-soon footer; pre-fills the review scene's first round note (your typed note wins).
weekly ledger
The 2048×2048 stat post fills week number, elo, weekly delta, k/d, hs% by itself — open the Social Kit, screenshot, post. Weekly delta baselines from the first open of each ISO week.

Notes: deltas are stored in the browser source's local storage — they survive OBS restarts. The API key is a client-side key baked into the scenes; anyone who sees it can spend your (free) quota, so keep the tweaks panel off stream. Success/failure logs to the browser console as [faceit].

02 — real alerts via streamelements (config, ~15 min)
  • streamelements.com → sign in with Twitch → Streaming Tools → AlertBox → copy the overlay URL into an OBS browser source (in the shared "all" nested scene, top-center, y=140).
  • Per alert type (follower / sub / raid / cheer): layout text only (no image, no gif), animation fade in/out, duration 4s, TTS off.
  • Text template: new follower — {name} (lowercase, matching the built-in demo alert).
  • Sound: upload exports/alert-tick.wav, volume ≈ 25% (the −18dB law).
  • Custom CSS (paste in the AlertBox's CSS tab — class names may drift with SE updates):
*, .widget-container { font-family: 'Courier New', ui-monospace, monospace; }
#alert-image, .alert-image { display: none !important; }
#alert-text, .alert-text-wrap {
  background: rgba(14,14,14,0.92); padding: 16px 28px;
  text-transform: uppercase; letter-spacing: 0.14em;
  font-size: 20px; color: #B9B9B4;
}
#alert-text .name, .alert-name, #alert-username {
  font-family: Helvetica, Arial, sans-serif;
  font-weight: 700; font-size: 26px;
  color: #FAFAF8; text-transform: none; letter-spacing: 0;
}

Once live, turn the built-in demo alert off: OBS 02 tweaks → alert → showAlert = off.

03 — chat overlay for non-gameplay scenes (config, ~5 min)
  • OBS browser source in scenes 03 / 04 only (never 02): URL https://www.twitch.tv/popout/mezcs_/chat, size 420×600, right edge, 96px margins.
